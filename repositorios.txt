Proyecto: Municipalidad Valle del Sol - Prevención de Incendios

Al ser un entorno de desarrollo estructurado como Monorepositorio, todo el código fuente se encuentra consolidado en el siguiente enlace principal. 

Enlace al repositorio principal del proyecto (Monorepo):
https://github.com/DarkoRodriguez/plataforma-incendios-valle-del-sol

A continuación, la descripción de lo que contiene cada directorio clave dentro del repositorio:

- /mfe-mapeo: Micro-frontend desarrollado en React con Vite y Leaflet. Responsable de mostrar el mapa interactivo y permitir el reporte de incendios.
- /ms-bff: Backend For Frontend (API Gateway) desarrollado con Spring Cloud Gateway. Actúa como orquestador y punto de entrada.
- /ms-mapeo: Microservicio en Spring Boot con PostgreSQL + PostGIS para la gestión espacial de los focos de incendio reportados.
- /ms-usuarios: Microservicio en Spring Boot responsable de la autenticación e información de usuarios/brigadistas.
- /archetypes: Arquetipo de Maven para la generación estandarizada de nuevos microservicios en la plataforma.
